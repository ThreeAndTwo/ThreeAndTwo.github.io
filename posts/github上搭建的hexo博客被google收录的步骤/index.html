Github 上搭建的 hexo 博客被 Google 收录的步骤

Tuesday, January 21, 2020

由于自搭建部署在 github 上的 hexo 博客并不会被搜索引擎收录，所以想让Google收录一下，至于百度，由于要手机号，放弃了。

配置前的工作

  1. 在部署 hexo 博客的根目录下安装插件
npm install hexo-generator-sitemap --save			// eg: /Users/kai/Code/hexoBlog

ps: 会产生 public/sitemap.xml 的文件。

  1. 配置 blog 最上级目录的 _config.yml 文件
#google search
sitemap:
  path: sitemap.xml			
## 注意次文件下的 url 务必是自己的 blog 的 url
  1. 编译并推送 blog
$ hexo g
$ hexo s

ps: 此时 sitemap.xml 文件有以前文章的信息了

配置 Google 收录博客

  1. 进入 Google 收录的地址 传送门

    ps: 选右边的那个提交 url 即可

  2. 设置站点地图

这样提交即可

补充

过了一天发现被收录了，但是可能直接在引擎里面搜索是发现不了的，因为 Google 第一步收录的是网页，再然后将其加入到 搜索的索引当中的。所以如果用 site:yoursite:github.io 搜索不到的时候可以再等等，或者手动提交加索引当中。如下图操作所示。
