Goland 使用技巧

Saturday, July 18, 2020

Goland 版本：2020.1，Go 夜读 Goland 视频整理笔记

1. 渔 – 快捷键查询

2. Switcher 窗口切换

​ IKBC Poker2：ctrl + tab

3. Quick Documentation

​ IKBC Poker2：Fn + 1

4. 调取 / 隐藏 左侧的 Project 区域（获得最大写代码的空间）

​ IKBC Poker：Win + 1

注意：当鼠标的焦点在代码区域时，快捷键需要按两次，第一次是将焦点放到左侧的 Project 上面，第二次则执行隐藏 Project 的操作。

5. Code Completion -> Basic

IKBC Proker：ctrl + space

注意

  1. 当鼠标放在不同位置时，Goland 智能提示会有变化！

  2. 如果 Mac 上安装了 Thing3，快捷键可能有冲突，可以选择更改 thing3 的快捷键或者关掉

6. Recent File - Show changed only

​ IKBC Poker：WIN + E

注意

  1. 连续按两次快捷键，选定复选框，则会按照最近修改文件的顺序排序

7. Recent Location - Show changed only

IKBC Poker：WIN + Shift + E

注意

  1. 如果 Mac 上安装了“搜狗拼音”，则快捷键会发生冲突！可选择将“搜狗拼音”的改选项设置为“禁用快捷键”或者更改“搜狗拼音”的改选项的快捷键。

8. 快速写代码

注意

  1. nn 代表 not nil
  2. 具体的补全，打.之后选择不同的选项就好，且有说明

9. 搜索 - Search Everywhere

​ ALL(IKBC Poker)：shift*2

注意

  1. 可以搜索工具相关的任何东西，快捷键，方法，文件等等。

10. 窗口快速查找

Goland 所有窗口都支持，指需要输入待查找的字符串（或关键字）即可

11. 快速修正代码中的错误

​ IKBC Poker：alt + enter

提示

  1. 修正有 2 中方式，一种使用鼠标去点击，一种使用快捷键
  2. 快捷键更快一些

12. 快速运行快捷键

IKBC Poker：ctrl + Shift + R

13. 反向快速补全代码

tips

  1. 全程 alt + enter

14. Goland 快速实现代码重构

tips

  1. 选取要重构的代码
  2. 调出 Goland 相关面板
  3. 修改方法名

修改重构方法参数的正确方式

15. 快速定位到下一处代码发生错误的位置

tips

  1. 或许有冲突，可以更改快捷键设置

  2. 使用 shift*2 搜索 next

    然后做相应的代码修正即可！

16. Extend Selection

​ IKBC Poker：alt + Fn + w

tips

  1. 按照代码区块选定，可以避免鼠标选择的遗漏
